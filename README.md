# Thulomanche

Frontend-only React + Vite site.

## What’s included

- Home page
- Unicode → Preeti converter tool (`/tools/unicode-to-preeti`)
- Styling + basic app routing

> Note: The backend and all backend-dependent admin/auth/post features have been removed.

## Tech Stack

- React 18
- React Router v6
- Vite
- Axios (if used by any remaining frontend code)
- lucide-react + react-quill (if used by components)

## Getting Started

### Prerequisites
- Node.js

### Run locally
```bash
npm install
npm run dev
```

Then open the URL printed in your terminal (typically http://localhost:3000).

## Deployment
Build the frontend:
```bash
npm run build
```
