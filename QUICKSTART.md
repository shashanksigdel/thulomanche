# Quick Start Guide

## 1) Install & Run (Frontend-only)

```bash
npm install
npm run dev
```

Then open the URL printed in your terminal (typically http://localhost:3000).

## Available Routes / Pages

- `/` – Home (Unicode → Preeti converter tool)
- `/tools/unicode-to-preeti` – Unicode → Preeti converter tool

## Next Steps (Optional)

- Configure any Vite environment variables you need (if used by the frontend).
- Deploy the frontend (Vercel / Netlify / static hosting).

---

Enjoy building! 🚀
