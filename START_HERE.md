## ✅ What you get (Frontend-only)

This repo is now **frontend-only** (React + Vite). The previous backend (Express/MongoDB) has been removed.

### Available routes
- `/` – Home (Unicode → Preeti converter tool)
- `/tools/unicode-to-preeti` – Same converter tool

## 🚀 Getting Started (Frontend)

### Step 1: Install dependencies
```bash
npm install
```

### Step 2: Start the dev server
```bash
npm run dev
```

### Step 3: Visit the app
Open the URL printed in your terminal (typically http://localhost:3000)

## 🐛 Troubleshooting

### App doesn’t start
- Confirm you’re running from repo root
- Run `npm install` again

### Missing modules / build errors
- Ensure dependencies were installed: `npm install`
- Re-run `npm run build`
